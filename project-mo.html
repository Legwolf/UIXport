Motion Prototypes: Framer Design & Code

Motion graphics and interactive design with Framer library.

OVERVIEW

These projects are personal experiments on animation, motion, and prototyping with Framer application. I wrote this series of motion graphics and interactive elements with very basics, simple properties, and commonly used patterns to present my ideas.

OBJECTIVE

Prototypes, JavaSript, interactive design, animation, motion graphics

ITEM

Design challenge

TOOLS

Framer, Sketch, Photoshop

PERIOD

Aug 2013 - Oct 2013

DESIGN TOOLS

Framer software programming process

I was using the library of Framer, an open-source JavaScript framework for both low and high fidelity prototyping. I designed the transition, interaction, and animation of elements in the user interface.
